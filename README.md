# calculator_project
## Description
This is a simple Calculator project built using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear (C)
- Delete (DEL)

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User-friendly interface
- Responsive design
- Fast calculations
- Error handling for invalid inputs

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Double-click on `index.html` or open it in any web browser.
